# Cookbook

Short, complete recipes that put the pieces together. Each one is a self-contained component you can drop into an app created with [Getting Started](../guide/getting-started).

- [Building a Counter](./counter) — state, key handling and quitting
- [A Bouncing Box](./bouncing-box) — animation with `useInterval` and absolute positioning
- [Forms & Inputs](./forms) — `<Input>`, `v-model`, submitting and a live progress bar

Looking for something to run? The repository's [`playground/`](https://github.com/vue-terminal/vue-termui/tree/main/playground) is a full multi-page app demonstrating every component, with a focusable sidebar — a great place to read real code.
