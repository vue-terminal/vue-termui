# create-vue-termui

Scaffold a new [vue-termui](https://github.com/vue-terminal/vue-termui) terminal app.

```bash
npm create vue-termui@latest
# or
pnpm create vue-termui
```

Pass a directory to skip the prompt:

```bash
npm create vue-termui@latest my-app
```
